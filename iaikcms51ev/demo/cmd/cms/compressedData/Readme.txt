This directory contains a cmd files for running demos
showing how to use the CMS content type CompressedData
for content compression.

The required libraries (iaik_jce.jar, iaik_cms.jar, 
iaik_cms_demo.jar) have to be located in the 
../../../lib directory. If you are using a JDK
version > 1.3, you will need the signed version of 
iaik_jce.jar located in the ../../../lib/signed directory.


CompressedDataDemo:
===================

Shows how to use the IAIK-CMS CompressedData and
CompressedDataStream implementations.

Usage:

Start the demo by running

   > runCompressedDataDemo.bat 


CompressedDataOutputStreamDemo:
==================================

Shows how to use the IAIK-CMS CompressedDataOutputStream
implementation.

Usage:

Start the demo by running

   > runCompressedDataOutputStreamDemo.bat
