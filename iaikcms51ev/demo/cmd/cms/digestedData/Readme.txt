This directory contains cmd files for running demos
showing how to use the CMS content type DigestedData
for protecting the integrity of some contents be means
of a Message Digest.

The required libraries (iaik_jce.jar, iaik_cms.jar, 
iaik_cms_demo.jar) have to be located in the 
../../../lib directory. If you are using a JDK
version > 1.3, you will need the signed version of 
iaik_jce.jar located in the ../../../lib/signed directory.



DigestedDataDemo:
=================

Shows how to use the IAIK-CMS DigestedData and
DigestedDataStream implementations.

Usage:

Start the demo by running

   > runDigestedDataDemo.bat 


DigestedDataOutputStreamDemo:
=============================

Shows how to use the IAIK-CMS DigestedDataOutputStream
implementation.

Usage:

Start the demo by running

   > runDigestedDataOutputStreamDemo.bat
