This directory contains a shell script for running basic
CMS demos showing the usage of all content type 
implementations (AuthenticatedData, CompressedData,
Data, DigestedData, EncryptedData, EnvelopedData,
SignedData).

The required libraries (iaik_jce.jar, iaik_cms.jar, 
iaik_cms_demo.jar) have to be located in the ../../../lib directory.
If you are using a JDK version > 1.3, you will need the signed 
version of iaik_jce.jar located in the ../../../lib/signed directory.


Usage:
======

Start the demo by running

   ./runCMSDemo.sh
