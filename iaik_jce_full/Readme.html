IAIK-JCE 5.3: Information

This package contains an implementation of the Java Cryptography Extension (JCE, specified by SUN/ORACLE) and other cryptographic classes.

The IAIK-JCE toolkit is a set of APIs and implementations of cryptographic functionality, including hash functions, message authentication codes, symmetric, asymmetric, stream, and block encryption, key and certificate management . It supplements the security functionality of the default Java JDK.

The architecture of the IAIK-JCE follows the same design principles found elsewhere in the JCA.
