IAIK JCE Provider for PKCS#11
=============================

This package contains a Java JCE provider for the 
integration of smart cards, hardware security modules 
and other external cryptographic modules. It uses the 
PKCS#11 standard API to interface with these modules. 
Any cryptographic device can be integrated which is 
compatible with the PKCS#11 version 2.20 specification or lower.

The current version of this package is available from

http://jce.iaik.tugraz.at/download/

After the installation has finished use your favorite 
browser to view the doc/Readme.html for further information.
